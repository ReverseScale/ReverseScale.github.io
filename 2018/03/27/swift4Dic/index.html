Swift 4.0 中对 Dictionary 的改进（转载）

#Swift 4 发布已经有一段时间了， 不知道大家有没有切换到 4.0 版本。 这次 4.0 更新给我最大的感受就是没有了前几次升级的跳跃式变化。

不用为了更新语言版本，完全推翻已有的项目，这也是 Swift 慢慢趋向于稳定的标志。

咱们这次说说 Swift 4.0 对 Dictionary 这个经常会用到的类的改进。
